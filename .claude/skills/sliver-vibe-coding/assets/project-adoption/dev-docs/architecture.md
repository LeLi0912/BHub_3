## 禁止路径

- 禁止把半路项目当空项目重写。
- 禁止 UI、controller、route 拥有核心业务规则。
- 禁止 mock/假数据冒充真实功能。
- 禁止为了速度保留两套 owner。
- 禁止没有用户确认就删除旧字段、旧接口、旧文档或旧部署脚本。
- 禁止没有证据就切换框架、SDK、数据库或语言。
