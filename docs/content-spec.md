# GenoWeekly — 内容格式规范

## 整体样式规范

| 属性 | 值 |
|------|-----|
| 风格 | 简报/期刊风格 (Newsletter/Briefing) |
| 背景色 | 白色 `#ffffff` |
| 辅助背景 | 浅灰 `#f8f9fa` |
| 文字色 | 深灰 `#1a1a2e` / `#333333` |
| 次要文字 | 中灰 `#6b7280` |
| 强调色 (Accent) | 橙 `#e86f2a` |
| 标题字体 | Serif (Georgia / "Noto Serif SC") |
| 正文字体 | Sans-serif (Inter / "Noto Sans SC") |

| 圆角 | 元素圆角 6px |

### 布局交互说明

| 组件 | 行为 |
|------|------|
| **领域卡片** | 点击跳转到对应领域的内容面板；空领域卡片不可点击（灰色） |
| **分布图** | 仅展示计数，不可点击；无更新领域显示为灰色细条 |
| **领域侧边栏** | 点击切换右侧内容面板；< 900px 时变为横向滚动标签栏 |
| **内容面板** | 显示选定领域的工具表格 + 论文表格；无内容时显示 "暂无更新" |

## 排版与写作规范

### 领域卡片规范
- 网格布局: `repeat(auto-fit, minmax(180px, 1fr))`，缺口 16px
- 卡片组成: 中文领域名 + 英文名（小字）+ 领域简介 + 状态徽章
- 状态徽章: 有更新 → 橙底 `#fff4eb` + 橙字 `#e86f2a` "N 条更新"；无更新 → 灰底 `#f3f4f6` + 灰字 `#9ca3af` "暂无更新"
- 点击行为: 有更新的卡片可点击，调用 `showDomain()` 切换到对应内容面板
- 空领域卡片: 不可点击（`pointer-events: none`），灰色样式

### 领域分布图规范
- 纯 CSS 实现（无 JS 图表库），使用 flex 布局 + `width: N%` 控制条长
- 每行: 色条 + 领域名 + 数字计数
- 色条: 渐变填充（`linear-gradient`），统一高度 24px，圆角右端
- 无更新领域: 灰色细条（宽度 4px），表示零数据
- 顶部显示总数: "共 XX 条更新"

### 领域侧边栏规范
- 桌面端 (>900px): 固定左侧边栏（~180px）+ 右侧内容面板（flex 布局）
- 平板端 (<900px): 侧边栏变为横向滚动标签栏（`overflow-x: auto`），置于内容区上方
- 当前选中项: 橙底白字
- 点击切换: 通过 JS 控制内容面板的显示/隐藏（`display: block/none`）

### 表格规范
- 表头背景色: 浅橙底色 `#fff4eb`，深橙文字 `#e86f2a`
- 表格边框: 1px solid `#e5e7eb`
- 行悬停效果: 浅灰 `#f9fafb` 背景
- 交替行背景: `#ffffff` / `#f9fafb`
- 移动端: 表格支持横向滚动

### 链接规范
- 链接色: 强调橙 `#e86f2a`
- 悬停色: 深橙 `#d1541f`
- 外部链接显示图标（可选）

### 文字规范
- 工具/论文名称: 加粗
- 描述文字: 常规字重
- 领域标签: 小号徽章样式（圆角背景块）

### 中英文混排
- 中文与英文之间加空格（由 Jekyll 插件或手动保证）
- 数字与单位之间加空格（如 "50 bp" 而非 "50bp"）
- 专有名词保持原始大小写（如 scRNA-seq, CRISPR-Cas9）
