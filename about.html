关于 GenoWeekly

什么是 GenoWeekly？

GenoWeekly 是一份每周发布的生物信息学资讯汇总，旨在帮助研究人员快速了解领域内的最新进展。每期内容包括：

  • 新工具与数据库更新 — 本周发布的新生信工具、软件和数据库
  • 新方法与论文 — 值得关注的方法学论文和研究成果
  • 前沿进展综述 — 编辑精选的本周重要进展和趋势分析

数据来源

GenoWeekly 的内容整合自以下公开数据源：

BioRxiv

生物科学预印本服务器，覆盖生物信息学、基因组学等分类的最新预印本。

biorxiv.org →

PubMed

NCBI 生物医学文献数据库，通过 E-utilities API 按领域关键词检索。

pubmed.ncbi.nlm.nih.gov →

GitHub Trending

GitHub 趋势页面，筛选生物信息学相关话题的周热门仓库。

github.com/trending →

内容分类

所有条目按以下 7 个生物信息学领域进行分类：

基因组学 转录组学 单细胞组学 蛋白质组学 表观遗传学 宏基因组学 时空组学

发布方式

GenoWeekly 是一个纯静态网站，由 Jekyll 构建并部署于 GitHub Pages。内容经自动化采集和人工审核后发布，确保信息质量和准确度。

项目源代码托管在 GitHub 上，欢迎通过 Issues 和 Pull Requests 贡献。
